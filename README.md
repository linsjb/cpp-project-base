Project base for simple cpp projects.

The Makefile auto add all .cpp files in /src folder. No additional adding is needed.

# Installation steps
- Clone
- Rename root folder
- Run `make build-structure`
- Rock on

# Make commands
- make
  - Compile the source
- run
  - Running the output (program)
- clean
  - Clean output file(s)
- compile-run
  - Compile *and* run the program
- build-structure
  - Build the structure
